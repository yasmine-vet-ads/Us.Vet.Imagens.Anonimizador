# Requisitos do MVP

## Nome do projeto

Us.Vet.Imagens.Anonimizador

## Objetivo

Desenvolver um MVP para anonimização de imagens ultrassonográficas veterinárias, removendo áreas que possam conter dados sensíveis antes de uso acadêmico, organização em banco de imagens ou compartilhamento.

## Requisitos funcionais

- Permitir upload de imagens.
- Permitir upload de múltiplas imagens.
- Exibir imagem original.
- Aplicar anonimização automática por região.
- Permitir escolha entre tarja preta, desfoque e pixelização.
- Exibir imagem anonimizada.
- Permitir download individual.
- Permitir download em lote.
- Exportar imagens sem metadados.

## Requisitos não funcionais

- Ser simples de executar localmente.
- Ter interface acessível.
- Utilizar Python.
- Não exigir banco de dados na primeira versão.
- Não armazenar imagens em servidor externo.
- Permitir revisão visual antes do uso da imagem.
